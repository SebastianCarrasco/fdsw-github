Acerca de mi

Ingeniero en informática con mención en desarrollo de sistemas, profesor de las carreras técnicas de la Universidad de Antofagasta y Soporte computacional para la minera Sierra gorda SCM.

Desarrollador

Desarrollador con amplia experiencia en la mejora de sistemas internos y gestión de datos. Participé en la migración de sistemas del Ferrocarril de Antofagasta-Bolivia, optimizando la eficiencia con PHP. Lideré el desarrollo del sistema 'Hoja de Vida' en SERVIU, mejorando la seguridad de datos laborales. En Opendat S.A., amplié software minero, mejorando la gestión operativa. En IT Consultores SPA, desarrollé un módulo para la gestión de amonestaciones y configuré una nueva mesa de ayuda. En Electrans, fundé el Departamento de Informática, creando sistemas para gestión de compras, control de cajas operacionales y certificación de instrumentos.

Profesor

Educador en la Universidad de Antofagasta, especializado en desarrollo profesional técnico. Imparto clases vespertinas en diversos módulos como Mantenimiento de Bases de Datos, Hardware y Software, Redes, y Seguridad en TI. He dirigido talleres y sesiones especializadas, fortaleciendo las habilidades técnicas de los estudiantes y mejorando su preparación profesional. Mi enfoque se centra en brindar una educación práctica y aplicada, enriqueciendo el perfil académico de los alumnos.

Soporte

Ingeniero Especialista de Soporte en TRES60, desde septiembre de 2023. Proporciono soporte técnico integral para Minera Sierra Gorda, gestionando usuarios, configurando hardware, y optimizando la productividad administrativa. Además, administro el inventario de hardware y software, asegurando una gestión eficiente y actualizada de los recursos tecnológicos de la empresa.

Mis Desafios

Desafio 1 - CV Responsivo.

Desarrollé un CV interactivo utilizando HTML y CSS para destacar mi trayectoria profesional y habilidades. El proyecto incluyó una estructura limpia y responsive, optimizada para dispositivos móviles. Incorporé secciones clave como experiencia laboral, educación y habilidades, con un diseño moderno y accesible. Este CV digital no solo facilita la visualización de mi perfil profesional, sino que también demuestra mi competencia en tecnologías web fundamentales.

Desafio 2 - Vinilos Seba

Desarrollé la página web "Vinilos Seba" utilizando HTML y CSS, enfocándome en un diseño limpio y atractivo para una tienda de vinilos. El sitio cuenta con un header elegante, un carrusel de ofertas de vinilos, una sección de productos con imágenes y detalles, y un formulario de suscripción estilizado. La estructura es completamente responsive, garantizando una experiencia de usuario óptima en dispositivos móviles y desktop. Este proyecto destaca mi habilidad para crear sitios web modernos y funcionales, alineados con las mejores prácticas de diseño web.

Desafio 3 - Dinosaurios.

Desarrollé la página web "Dinosaurios" utilizando HTML y CSS, enfocándome en un diseño moderno y educativo. La página incluye un header con el título "Dinosaurios", una imagen central destacada, y una sección de texto informativo sobre los dinosaurios. Además, cuenta con una sección de características visualmente atractiva, con iconos representativos, y una galería de imágenes. El sitio es completamente responsive, asegurando una presentación consistente en dispositivos móviles y desktop, destacando mi capacidad para crear experiencias web educativas y visualmente atractivas.

Desafio 4 - Cuppon.

Desarrollé la página web "SebaCupones" utilizando HTML, CSS y Bootstrap, siguiendo una metodología Mobile First. La página incluye un header con un menú de navegación responsive, un grid de productos con tarjetas que muestran imágenes, descripciones y precios con descuentos destacados. Además, implementé una sección de suscripción con un formulario estilizado para capturar nombre y correo electrónico, y un footer que incluye el logo de la página y el texto de copyright. Este proyecto destaca por su diseño limpio, funcionalidad responsiva y uso efectivo de estilos para mejorar la experiencia del usuario.
